# **App Name**: Task Compass

## Core Features:

- Hourly Task View: Display tasks broken down by the hour for detailed scheduling.
- Daily Task View: Show all tasks due on a specific day.
- Weekly Summary: Provide a summary of all tasks scheduled for the current week.
- Daily Progress Review: A dedicated page for users to review completed tasks and track progress for each day.
- Weekly Progress Review: Allow users to see an overview of their achievements and workload over the past week.
- Monthly Progress Review: A page showing monthly task completion trends and insights.
- Unfinished Tasks List: Automatically list all overdue and incomplete tasks, so the user can prioritize what needs immediate action.

## Style Guidelines:

- Primary color: Calming blue (#79A8D3) for focus and clarity.
- Background color: Soft, off-white (#F2F3F7) to reduce eye strain.
- Accent color: Muted orange (#C99354) to highlight important actions.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern clarity and warmth.
- Use simple, geometric icons to represent different task categories.
- Prioritize a clean, single-column layout for easy task management.
- Subtle transitions to indicate task completion and updates.
